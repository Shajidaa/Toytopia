# ToyTopia

---

## _Where Every Toy Tells a Story!_

## Description:

**ToyTopia** is an engaging and responsive toy marketplace designed for kids and parents. It offers a seamless browsing experience where users can explore a variety of toys, view detailed information, and enjoy interactive features. The project focuses on delivering a joyful and user-friendly interface built with modern web technologies.

## Technologies used :

-**Core:** HTML5,CSS3,JavaScript

-**Framework:** Tailwind Css

-**Library:** React , React Router ,Recharts,React-Toastify

**package:**Swiper,
